Notes ////////////////////////////////////

Always leave the link to pivotx_essentials.css intact. It gets updated with PivotX. You can make adjustments to it (like i did) in style.css (in the 'earthlycalm' folder. Adjustments made will override the styles in pivotx_essentials.css.

Widgets ////////////////////////////////////
If you use widgets, set the widget style in Pivot > Extensions > (widgetname) to 'minimally styled'.

Delicious bookmarks:

I removed the tags that I used to bookmark a page. Change/cut and paste the following in Pivot/Administration//Widgets/Delicious

Widget style: choose minimally styled
Header format: <h3>My latest Del.icio.us bookmarks</h3><ul>
Output format: <li><a href="%link%" title="%description%">%title%</a></li>
Footer dormat: </ul>

Lifestream widget:

Change/cut and paste the following in Pivot/Administration//Widgets/Lifestream widget

Widget style: choose minimally styled
Header format: <h3>What I'm doing:</h3><ul>
Output format: <li>%title% - <span class='date'><a href="%link%">%date%</a></span></li>
Footer dormat: </ul>
